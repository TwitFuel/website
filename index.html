How to use

Send a message to the TwitFuel bot on telegram and click start



You will be prompted to link your twitter account.





After you click Link, you will be given a code.
Post a new tweet with the code and click I Have Tweeted



You will be asked to send the link for the tweet



After tweeting the code and sending the link, you will get a confirmation message.
Congratulations, you are now verified!
